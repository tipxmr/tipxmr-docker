# TipXMR Docker

Dockerize TipXMR for development and production

## Installation

### Dependencies

- git
- docker
- docker-compose

```bash
git clone https://github.com/tipxmr/tipxmr-docker.git
cd scripts
./install_linux.sh
docker-compose up
```

Then open [http://localhost:8080](http://localhost:8080) in your browser.
